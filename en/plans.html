Questions you might have

Which payment methods are accepted?

All common payment methods are accepted.

How can I cancel my subscription?

You can cancel your subscription at any time with a simple REST call and continue to use it in full until the end of the month.

Why is there only so low storage space offered?

The storage is located entirely in the server's main memory and can therefore be accessed as quickly as possible. This type of memory is expensive and is particularly suitable for use cases that require fast performance but do not need large storage space. Of course, your data is stored on permanent storage media in the background and additionally secured with daily backups.

What are request eqivalents and why are they limited?

In principle, there is no hard request limit. However, if there is an unusually high server load, it must be ensured that the performance are distributed fairly. In such a case, requests from users that consume a lot of resources and are above their request limit may remain unanswered. The request limit is expressed in request equivalents because not every request requires the same amount of performance. Four request equivalents roughly correspond to a simple GET request from a single resource. The request equivalent table shows the most important data.
